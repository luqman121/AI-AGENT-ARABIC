Professional Arabic-capable PDF creation and manipulation with page QA.

## Instructions

[مهارة: استوديو PDF] أنشئ/اقرأ/ادمج/قسّم/أعد ترتيب/دوّر صفحات PDF، وأضف ترويسات وتذييلات وأرقام
صفحات وعلامات مائية عند الطلب، واملأ النماذج المدعومة، واستخرج نصاً وجداول وصوراً، وحوّل المستندات
المدعومة إلى PDF، وادعم RTL العربي. جودة الإنشاء: غلاف احترافي عند المناسبة، تسلسل هرمي واضح، هوامش
مقروءة، ترويسة/تذييل متسقة، أرقام صفحات، فواصل أقسام ذات معنى، جداول مقروءة مع تكرار ترويسة الجدول
عبر الصفحات، تجنّب العناوين اليتيمة والقصّ العربي، وتحميل/تضمين الخطوط المطلوبة وفق تراخيصها (Cairo
أو خط عربي معتمد). لا تملأ كل صفحة بالزخارف، ولا تُنتج مقالب نص عادي وتسمّيها PDF احترافياً. التحقق:
أعد فتح الملف برمجياً، وأكّد عدد الصفحات وسلامة الملف، واعرض كل صفحة كصور، وافحص
القصّ/التداخل/الحروف المفقودة/تشكيل العربية/الصفحات الفارغة/تجاوز الجداول، وشغّل تمريرة إصلاح عند
وجود أخطاء، وأكّد وجود الملف النهائي في التخزين قبل إعادة الرابط.
